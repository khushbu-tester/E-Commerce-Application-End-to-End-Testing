#  E-Commerce Application - End-to-End QA Portfolio

This project demonstrates a professional-grade testing lifecycle for a high-traffic E-commerce platform. It covers everything from **Master Planning** to **Mobile UI/UX** and **Cross-Browser** validation.

---

###  Repository Structure
- **[01_Test_Plan_and_Strategy](./01_Test_Plan_and_Strategy):** Master document defining the scope and tools.
- **[02_Mobile_App_Testing](./02_Mobile_App_Testing):** Specialized scenarios for Android and iOS.
- **[03_Cross_Browser_Report](./03_Cross_Browser_Report):** Validation on Chrome, Safari, and Firefox.
- **[04_API_Testing_Postman](./04_API_Testing_Postman):** API validation for Search and Cart.

---

### Key Testing Highlights
- **Mobile First:** Tested app behavior during incoming calls and low battery.
- **Cart Persistence:** Verified that items remain in the cart after logout.
- **Compatibility:** 100% UI alignment on Chrome (Windows) and Safari (iOS).
